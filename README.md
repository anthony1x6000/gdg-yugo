# Website Guessr

Website Guessr is an interactive web-based game where players test their knowledge of the internet's most popular layouts.

## Features

- **Core Gameplay Loop:** Players are shown an anonymized website layout and must guess which popular platform it belongs to.
- **Scraper:** A simple backend function that returns anonymized HTML of a site for guessing.

## Tech Stack

### Frontend
- **Framework:** React
- **Routing:** TanStack Router
- **State Management:** Zustand
- **Data Fetching:** TanStack Query
- **Language:** TypeScript

### Backend
- **Serverless Compute:** Cloudflare Workers
- **Language:** TypeScript

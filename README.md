# Website Guessr

Website Guessr is an interactive web-based game where players test their knowledge of the internet's most popular layouts. The game presents users with a website that has been stripped of all its defining visual characteristics—such as logos, images, icons, and text. The challenge? Identify the website from four multiple-choice options based purely on its structural layout. 

## Features

- **Core Gameplay Loop:** Players are shown an anonymized website layout and must guess which popular platform it belongs to (e.g., YouTube, Google, Twitter).
- **Leaderboard:** Compete against other players globally to secure the highest score.
- **User Authentication:** Streamlined signup and login system. No email required! Users simply provide credentials and an optional profile picture URL.

## Tech Stack

This project is built using modern, type-safe web technologies:

### Frontend
- **Framework:** React
- **Routing:** TanStack Router
- **State Management:** Zustand (for Auth / Global State)
- **Data Fetching:** TanStack Query
- **Language:** TypeScript

### Backend & Database
- **Serverless Compute:** Cloudflare Workers
- **Database:** Cloudflare D1 (Serverless SQLite)
- **ORM:** Drizzle ORM
- **Language:** TypeScript
